## 15. Translation Guidelines

### General Rules

1. **Consistency**: Always use the same Russian term for the same English term
2. **Technical Terms**: Keep English for technical terms in logs/debug (Buffer, Circuit Breaker, Heartbeat)
3. **Abbreviations**: Preserve technical abbreviations (Sph, Cyl, Add) but translate descriptions
4. **Natural Language**: Use natural Russian phrasing, not literal translation
5. **Context Matters**: Same English word may have different translations depending on context

### Specific Cases

**"Order" Translation:**
- Manufacturing Order → Заказ на изготовление
- Sales Order → Заказ клиента
- Sort Order → Порядок сортировки

**"Sale" vs "Sales":**
- Sale (single) → Продажа
- Sales (general) → Продажи
- Sales Team → Отдел продаж

**"Stock" Translation:**
- Stock (inventory) → Остатки / Складской учёт
- Stock (warehouse) → Склад
- In Stock → В наличии

### Formality Level

- Use formal "Вы" for customer-facing messages
- Use informal "ты" only in internal debug/developer messages
- Prefer shorter forms for button labels
- Use full forms for help text and documentation
